# Stablecoin Analytics with Dune

This repository contains a Dune Analytics dashboard project analyzing USDC treasury use cases for enterprise operations.

## Current Project

### USDC Treasury Analysis for Fortune 500 Operations

This project evaluates whether USDC can support enterprise-level treasury and payment operations by analyzing transaction capacity, transaction size distribution, estimated fee savings, stress-period classification, and network performance during volatile periods.

Live Dashboard: https://dune.com/badm_16/treasury-analysis-for-fortune-500-operations-based-on-usdc

## Tools

- Dune Analytics
- Structured Query Language (SQL)
- Dashboard reporting
- Blockchain transaction data
- Business intelligence documentation
