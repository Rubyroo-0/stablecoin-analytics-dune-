# USDC Treasury Analysis for Fortune 500 Operations

## Project Overview

This project analyzes whether USD Coin (USDC) can support enterprise-level treasury and payment operations for large organizations such as Fortune 500 companies.

The dashboard evaluates transaction capacity, transaction size distribution, estimated cost savings compared with bank wire fees, and network performance during stress periods.

Live Dashboard: https://dune.com/badm_16/treasury-analysis-for-fortune-500-operations-based-on-usdc

## Business Question

Can the USDC network provide enough scale, stability, and cost efficiency to serve as a potential payment rail for enterprise treasury operations?

## Dashboard Sections

### 1. Daily Processing Capacity

Analyzes USDC daily transaction activity over time to evaluate whether the network is large enough and stable enough for enterprise-level treasury operations.

### 2. Transaction Size Distribution

Groups USDC transactions into size-based proxy segments:

- Retail
- Small Business
- Mid-Market
- Institutional
- Large Institutional

These categories are proxy labels based on transaction size buckets, not confirmed user identities.

### 3. Savings vs. Bank Wire Fee

Compares estimated USDC transaction costs with traditional bank wire transfer costs to evaluate potential cost-saving opportunities.

### 4. Stress Period Classification

Classifies recent days into normal, moderate-stress, and high-stress periods based on transaction activity and volatility-related behavior.

### 5. Network Performance During Stress

Analyzes whether USDC transaction activity remains stable during demand spikes or volatile periods.

### 6. SVB Crisis Case Study

Compares a normal baseline period with the Silicon Valley Bank crisis week in March 2023 to evaluate how USDC behaved during a real-world financial stress event.
